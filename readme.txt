Sentiment Analysis on IMDB reviews for ‘Game of Thrones’ tv show. (NLP, Python, Decision tree, Naïve Bayes, Random Forest, Tableau)
•	Extracted user comments and ratings from IMDB official website using python web scraping.
•	Performed data cleaning and pre-processing steps like stemming and tokenization and applied bag of words model.
•	 Classified the reviews as positive and negative by applying Random Forest algorithm. Also applied Naïve Bayes and Decision tree classification models to compare the efficiency.
